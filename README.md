## CoCode-iOS
CoCode编程交流社区 iOS 客户端

## Usage
* ``cd`` the code path
* ``pod install --no-repo-update`` in your terminal to install required pods
* open ``CoCode.xcworkspace`` and run
* use your iPhone instead of simulator as which is not supported for QQ SDK
